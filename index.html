Welcome to the Dummy API Restaurants

This is a simple API that provides restaurant information.

Available Endpoints

GET /restaurants

Returns a list of all available restaurants.

https://api-restaurant-ol2j.onrender.com/restaurants

GET /restaurants/:id

Fetches details for a specific restaurant by its :id.

https://api-restaurant-ol2j.onrender.com/restaurants/rqdv5juczeskfw1e867

GET /restaurants/search/:category

Fetches a list of restaurants that belong to a specific :category.
